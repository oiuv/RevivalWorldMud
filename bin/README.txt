====================================================================

因為之前的版本有些東西還是會出現問題, 因此再次修改後 Release 新版本

mysql 相關 efun 請自行上網查閱

====================================================================
簡易安裝步驟
====================================================================

第一步
  請先把本壓縮檔解開放到 C:\mud 下 ( 亦可直接把本解壓縮檔放到 c:\ )
  同時將 RWlib 也同樣解壓縮到 C:\mud\lib 下

  此時在 C:\mud\ 裡面就會有
  1. config.rw
  2. v22.2b14 RW-build.exe
  3. lib

第二步
  執行 v22.2b14 RW-build 之後後會出現一個視窗
  Mudlib 的配制文件請輸入 c:\mud\config.rw
  地址服務器端口請輸入 5990

第三步
  壓下確定, 請使用你的 client 連線到 127.0.0.1 5000 創造你的新人物
